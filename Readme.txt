# MCQ System (Multiple Choice Question System)

## Project Overview

The **MCQ System** is a simple console-based application designed to conduct multiple-choice quizzes. It allows users to answer questions, validates their responses, and calculates the final score. This project is ideal for beginners to understand core programming concepts and basic application flow.

---

## Features

* Display multiple-choice questions
* Accept user input
* Validate answers
* Calculate and display final score
* Simple and user-friendly console interaction

---

## Technologies Used

* **Programming Language:** Java
* **Concepts Used:**

  * OOP (Classes & Objects)
  * Arrays / Collections
  * Conditional statements
  * Loops
  * User input handling (`Scanner`)

## How to Run the Project

1. Clone the repository or download the source code.
2. Open the project in any Java IDE (Eclipse, IntelliJ, VS Code).
3. Compile the Java files:

   ```
   javac *.java
   ```
4. Run the main class:

   ```
   java Main
   ```

---

## How It Works

1. The system displays a question with multiple options.
2. The user selects an option.
3. The system checks whether the selected option is correct.
4. The score is updated accordingly.
5. After all questions, the final score is displayed.
